Github

o que é o Github

O Github vem do Git criado por Linus Torvalds onde seu objetivo era um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005.

Para que serve

Ele serve para você salvar seu codigo na nuvem e manter atalizado sempre que for preciso.

Como Funciona

hospeda repositórios do Git e fornece aos desenvolvedores ferramentas para enviar um código melhor por meio das funcionalidades de linha de comando, problemas(discussões encadeadas), pull requests, revisão de código ou o uso de uma coleção de aplicativos grátis e para compra em GitHub Marketplace.
